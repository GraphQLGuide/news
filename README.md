### Setup

```
git clone git@github.com:GraphQLGuide/news.git
cd news
npm install
npm start
```

App: [127.0.0.1:3000](http://127.0.0.1:3000/) (Github OAuth doesn't work on localhost)

GraphiQL: [/graphiql](http://127.0.0.1:3000/graphiql)
